# React 会計ソフト

日本の中小企業・個人事業主向けのシンプルな会計アプリケーションです。

## 概要

ブラウザ上で動作するSPA（Single Page Application）型の会計ソフトです。サーバーやデータベースを必要とせず、ブラウザのLocalStorageにデータを保存します。

## 主な機能

### 仕訳・帳簿
- **仕訳入力** - 借方/貸方の仕訳登録、テンプレート保存
- **元帳** - 勘定科目別の取引履歴
- **補助元帳** - 補助科目別の取引履歴

### レポート
- **試算表** - 合計残高試算表の作成
- **財務諸表** - 貸借対照表(B/S)、損益計算書(P/L)
- **CF計算書** - キャッシュフロー計算書
- **月次推移** - 月別の収益・費用推移グラフ
- **得意先残高** - 取引先別の残高管理
- **消費税** - 消費税計算・レポート

### マスタ管理
- **勘定科目** - 科目の追加・編集（資産/負債/純資産/収益/費用）
- **補助科目** - 補助科目の管理
- **固定資産** - 固定資産台帳、減価償却計算

### その他
- **期首残高** - 期首残高の設定
- **年度締め** - 繰越処理
- **バックアップ** - JSONでのエクスポート/インポート
- **印刷/PDF出力** - 各帳票の印刷・PDF保存

## 技術スタック

- **React 18** + **TypeScript**
- **Vite** - ビルドツール
- **Zustand** - 状態管理
- **TanStack Table** - テーブル管理
- **Recharts** - グラフ描画
- **Tailwind CSS** - スタイリング
- **LocalStorage** - データ永続化

## サンプルデータで操作を確認

アプリの機能を試すためのサンプルデータが用意されています。

### サンプルデータの読み込み方法

1. アプリを起動し、「バックアップ」タブを開く
2. 「サンプルデータを生成」ボタンをクリック

### サンプルデータの内容

- **仕訳データ**: 約30件の仕訳（4〜6月の3ヶ月分）
  - 売上、仕入、経費、給与、固定資産購入など
  - 様々な消費税区分（課税売上、課税仕入、非課税など）
- **勘定科目**: 基本的な勘定科目と補助科目
- **期首残高**: 資産・負債・純資産の期首残高
- **固定資産**: パソコン、オフィス家具などの固定資産台帳

### 確認できる機能

| 機能 | 確認内容 |
|------|----------|
| 元帳 | 各勘定科目の取引履歴と残高推移 |
| 補助元帳 | 得意先別・仕入先別の明細 |
| 試算表 | 借方・貸方の合計と残高 |
| 財務諸表 | 貸借対照表(B/S)、損益計算書(P/L) |
| CF計算書 | キャッシュフローの計算結果 |
| 月次推移 | 月別の収益・費用グラフ |
| 消費税 | 課税売上・課税仕入の集計 |
| 固定資産 | 減価償却費の自動計算 |

> **注意**: サンプルデータをインポートすると、既存のデータは上書きされます。必要に応じて先にバックアップを取ってください。

## 使い方

### 1. 勘定科目の設定
「勘定科目」タブで必要な科目を追加・編集します。デフォルトで基本的な科目が登録されています。

### 2. 期首残高の設定（必要な場合）
「期首残高」タブで各科目の期首残高を設定します。

### 3. 仕訳入力
「仕訳入力」タブで日々の取引を入力します。よく使う仕訳はテンプレートとして保存できます。

### 4. レポートの確認
試算表、財務諸表、月次推移などで経営状況を確認できます。

### 5. バックアップ
定期的に「バックアップ」タブからデータをエクスポートしてください。

## 注意事項

- データはブラウザのLocalStorageに保存されます
- ブラウザのデータを削除すると仕訳データも消えます
- 定期的にバックアップを取ることを推奨します
- 本番環境での使用前に十分なテストを行ってください
